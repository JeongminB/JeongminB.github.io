## ◆ Bessel's correction

<a href="/basics/statistics/sample-mean-var">이전 글</a>에서 다룬 표본분산을 불편 추정량으로 만들어주기 위해서 $n-1$로 나눈다는 것을 수식적으로는 받아들일 수는 있겠으나, 그래도 $n$으로 나누면 안 되는 좀 더 와닿는 이유가 필요할지도 모르겠습니다. 그래서 이번에는 표본분산을 좀 더 자세히 들여다보고자 합니다.

$$ \sigma^2 = V[X] = \mathbb{E}[(X-\mu)^2] = \frac{\sum_{i=1}^n (X_i - \mu)^2}{n}$$

모집단의 분산은 확률변수가 아니라 고정된 상수입니다. 그러나 모집단의 크기가 커서 관찰된 샘플들에 대해서만 따질 수 있는 경우, 우리는 표본으로부터 모분산을 추정해야 합니다. 이때, 표본은 모집단에서 '무작위'로 뽑은 것으로 가정하기 때문에 여기서 계산되는 분산은 확률변수가 됩니다. 매 샘플에 따라 값이 달라질 것이기 때문입니다.

또한, 기존에 분산을 구할 때 필요한 모평균 $\mu$은 알 수 없기 때문에 표본평균 $\bar X$로 대체하게 되는데, 이 경우 추정량 $\hat \theta$의 편의 (bias), $\mathbb{E}[\hat \theta]-\theta$ 가 0이 아니게 됩니다. 즉, 표본분포의 분산 (편차제곱합을 $n$으로 나눈 값)을 추정량 $\theta$로 쓸 경우 모수(모분산) $\theta$를 잘 추정하지 못합니다. 그래서 $n-1$로 대신 나누어 표본분산을 불편 추정량으로 만드는데, 이러한 방법을 <strong><font color="#3367E5">베셀 보정</font></strong> (Bessel's correction)이라고 합니다.

## ◆ 표본분포의 분산 기댓값은 모분포 분산보다 작다

다시 돌아와서, 그러면 왜 $n$으로 나누면 값을 잘 추정하지 못하는 걸까요? 간단한 예시를 들어봅시다. 모집단의 평균이 `200`, 분산이 `100`인데 운이 좋게도(?) 아래와 같은 샘플들이 뽑혔다고 합시다. 

```
[177.15  171.78  169.01  172.32  174.13]
```

모분산은 '전체 표본들이 모평균에서 퍼진 정도'입니다. 예시에서 모평균 $\mu$는 `200`으로 고정된 값입니다. 그러나, 우리가 관찰할 수 있는 건 '일부 표본'과 그들의 '표본평균' 뿐입니다.

위에서 표본평균 $\bar X$는 대략 `172.88`입니다. 보다시피 $\bar X$는 표본에 따라서 달라지는 확률변수이며, 모평균을 중심으로 값이 한쪽으로 치우치게 됩니다 (위는 좀 극단적이지만요). 고로, '관찰된 표본들이 표본평균에서 퍼진 정도'를 표본분산으로 정의하면 추정하고자 하는 모분산과 차이가 발생할 것입니다.

$$ \mathbb{E}\left[\frac{\sum(X_i - \bar X)^2}{n}\right] \;<\; \mathbb{E}[(X - \mu)^2] = \sigma^2 $$

좀 더 구체적으로, 표본분포의 분산 '기댓값'이 만약 $\mathbb{E}[\frac{\sum(X_i - \bar X)^2}{n}]$이라면, 추정하고자 하는 모분산 $\mathbb{E}[(X - \mu)^2]$보다 작은 값을 가지게 됩니다. 즉, 표본분산을 단순히 표본분포의 분산으로 정의하면 편의 추정량이 됩니다.

이것의 이유는, 매 표본평균이 표본들의 중심값이 되기 때문에, 그리고 그 표본평균은 분산이 0이 아니기 때문에, '표본들이 모평균에서 퍼지는 정도'의 일부가 표본평균의 분산으로 흡수된다고 생각해도 좋을 것 같습니다. 혹은, '표본들이 표본평균에서 퍼지는 정도'를 구하면 관찰된 표본들이 '모평균에 대해서 치우침'이라는 정보를 잃는다고 봐도 좋지 않을까요? 

$$ \mathbb{E}\left[\frac{\sum_{i=1}^n(X_i - \bar X)^2}{n}\right] = \frac{n-1}{n} \sigma^2 $$

증명은 생략하겠습니다만 (하단의 링크 참고 바람), 위와 같이 표본분포의 분산 기댓값은 모분산보다 작은 값이 나옵니다. 기댓값이기 때문에 항상 작은 값을 추정하는 것은 아닙니다. 위 예시에서 5개의 샘플이 모평균 대비 크고작은 값이 나와버리면 모분산 `100`보다 큰 분산 값이 나올 수도 있습니다 (표본의 수가 적으므로 확률은 희박하겠지만요).

$$ \mathbb{E}\left[\frac{\sum(X_i - \bar X)^2}{n}\right] = \sigma^2 - \frac{\sigma^2}{n} = V[X] - V[\bar X] $$

역으로, 모분산은 표본분포의 분산에 표본평균의 분산을 더한 것과 같습니다.

<div>

$$ \require{cancel} \mathbb{E}[s^2] = \mathbb{E} \left[ \frac{\cancel{n}}{n-1}  \frac{1}{\cancel{n}} \sum_{i=1}^n (X_i - \bar X)^2 \right] = \sigma^2 $$

$$ s^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i - \bar X)^2 $$
</div>

표본분산 $s^2$를 $n$ 대신 $n-1$로 나누는 이유는 이정도로 충분할 것 같습니다.

## ◆ Testing on Google Colab

수식적 증명 대신 Google Colab에서 간단한 코드를 작성하여 테스트 해보겠습니다. 테스트의 조건은 아래와 같습니다.

* $N(200, 10^2)$ 분포에서 랜덤 추출한 10,000개의 샘플을 모집단 $X$로 가정.
* $X$에서 표본을 랜덤 추출한다. 표본의 수는 2개에서 30개까지 늘리면서 테스트.
* 테스트마다 표본분산($n$으로 나눈 것, $n-1$로 나눈 것), 표본평균의 분산을 구한다.
* 표본분산의 기댓값을 근사하기 위해 10,000번 시행 결과에 대한 평균을 낸다.
* 표본평균의 분산은 10,000번 시행을 통해 얻은 표본평균 값들에 대한 분산으로 근사한다.

```python
import numpy as np
import matplotlib.pyplot as plt

mu, sigma = 200, 10 
total_num = 10000
data = np.random.normal(mu, sigma, total_num)

real_mean = np.sum(data) / total_num
real_var = np.sum((data - real_mean)**2) / total_num

it = 10000
e_smv_list, e_sv1_list, e_sv2_list = [], [], []
start, end = 2, 30
X = range(start, end+1)
for n in X:
    smv, sv1, sv2 = [], [], []
    for _ in range(it):
        samples = np.random.choice(data, n)
        m_sample = np.sum(samples) / n
        devsq = np.sum((samples - m_sample)**2)
        smv.append(m_sample)
        sv1.append((devsq / n))
        sv2.append((devsq / (n-1)))
    smv = np.array(smv)
    sv1 = np.array(sv1)
    sv2 = np.array(sv2)
    e_smv = np.sum((smv - np.mean(smv))**2) / it
    e_sv1 = np.sum(sv1) / it
    e_sv2 = np.sum(sv2) / it
    e_smv_list.append(e_smv)
    e_sv1_list.append(e_sv1)
    e_sv2_list.append(e_sv2)

print("real var:", real_var)
print("num of iter:", it)
plt.rcParams["figure.figsize"] = (8,10)
plt.rcParams["legend.fontsize"] = 18

x = np.linspace(start, end, num=500)
y_smv = (x-1) / x * real_var
sum_esv_esmv = [sum(x) for x in zip(e_sv1_list, e_smv_list)]

x1, y1 = [start, end], [real_var, real_var]
plt.plot(x1, y1, 'k:')
plt.plot(x, y_smv, 'k--')
plt.plot(X, e_sv2_list, lw=3)
plt.plot(X, e_sv1_list)
plt.plot(X, e_smv_list)
plt.plot(X, sum_esv_esmv, '.-')
plt.legend(('real var', 'sample var',
    '$\\mathbb{E}\,[\\frac{\\sum(X_i - \\bar X)^2}{n-1}]$',
    '$\\mathbb{E}\,[\\frac{\\sum(X_i - \\bar X)^2}{n}]$',
    '$V\,[\\bar X]$',
    '$\\mathbb{E}\,[\\frac{\\sum(X_i - \\bar X)^2}{n}] + V\,[\\bar X]$'),
    loc='center right', shadow=True)
plt.xticks(np.arange(start, end+1, step=2))
plt.yticks(np.arange(0, 105+1, step=10))
plt.grid(True)
plt.show()
```

그래프를 보면, 편차제곱합을 $n$으로 나누면 $\frac{n-1}{n} \sigma^2$ `sample dist. var`를 추정하고, $n-1$로 나누어야 $\sigma^2$에 근접한다는 것을 확인할 수 있습니다. 각 표본 추출 테스트의 시행 회수 `num of iter` 를 늘리면 기댓값들이 이상적인 값인 점선에 가까워집니다 (x축과 무관함). 여기서는 각 line이 구분될 수 있게 적당한 수를 취했습니다.

또한, $n$으로 나누었을 때에도 추출하는 샘플의 수가 커지면 (x축 오른쪽으로 갈수록) 점차 모수에 가까워지는 것도 마찬가지로 확인할 수 있습니다.
